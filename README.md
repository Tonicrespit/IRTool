# IRTool

# Dependencies

R 3.0 or higher with the packages:
- ltm

pip3 install pandas
pip3 install numpy
pip3 install dash
pip3 install rpy2==2.9.5
pip3 install tzlocal
pip3 install pandas_ml
pip3 install xlrd

# How to use

Run using `python main.py` (or `python3 main.py`) and then go to http://localhost:8050
